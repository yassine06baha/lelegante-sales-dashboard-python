# L'Élégante - Sales Dashboard Python

Dashboard de suivi des ventes et du stock pour une boutique de mode féminine.

## Stack

- Python + Flask
- SQLite / SQL
- HTML / CSS
- JavaScript

## Fonctionnalités

- Création automatique d'une base SQLite avec produits, ventes et stocks
- API JSON pour exposer les indicateurs
- Dashboard web avec cartes KPI, tableau des produits et graphique des ventes
- Alerte visuelle sur les articles à faible stock

## Lancer le projet

```bash
pip install flask
python app.py
```

Puis ouvrir `http://127.0.0.1:5000`.

## Niveau

Intermédiaire : routing Flask, requêtes SQL agrégées, API REST, rendu dynamique côté navigateur.
